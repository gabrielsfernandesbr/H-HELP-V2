# H-HELP V2
## Swiss army knife for hackers and investors

## FOR LINUX UBUNTU/DEBIAN

### CLONE
```
git clone https://github.com/gabrielsfernandesbr/H-HELP-V2.git
```

### FOLDER
```
cd H-HELP-V2
```

### REQUIREMENTS
```
pip install -r requirements.txt
```

### CHMODE
```
sudo chmod +x main.py
```
### RUN
```
sudo ./main.py
```
